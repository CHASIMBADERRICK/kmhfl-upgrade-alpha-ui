## How to use

- Clone this repository ```git clone https://github.com/benzerbett/kmhfl-mbele-ui mfl3``` 
- cd into the folder ```cd mfl3```
- Install dependencies ```npm install``` or ```yarn```
- For local development, run ```npm run dev``` or ```yarn dev```
